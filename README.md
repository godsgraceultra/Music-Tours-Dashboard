# Music Tours Analysis & Dashboard

## Overview
This project analyzes historical and recent high-grossing music tours, focusing on revenue, show volume, and artist performance. The dataset was scraped from Wikipedia and cleaned using **Python (Pandas)** for accuracy and consistency.

## Tools Used
- **Python (Pandas):** Data cleaning, transformation, and preprocessing
- **Power BI:** Interactive dashboards and visualizations
- **Power Query & DAX:** Calculations and metrics in Power BI
- **Google Slides & Docs:** Report presentation and storytelling

## Features
- Cleaned and formatted messy dataset
- Calculated metrics like average revenue per show
- Visualized top tours and artists by revenue and number of shows
- Dashboard includes revenue trends by shows category and top artist cards

## Usage

1. Download the cleaned dataset (`cleaned_dataset.csv`) for analysis or practice.
2. Explore the data cleaning steps in the provided Jupyter Notebook (`Data Cleaning.ipynb`).
3. View the final report with visualizations and insights in the PDF (`Music Tours Dashboard Report.pdf`, `Music Tours Dashboard.pdf`).
4. Open the interactive Power BI dashboard in the PBIX file (`Music Tours Dashboard.pbix`) to explore the visuals and DAX calculations.
